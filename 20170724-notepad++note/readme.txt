*notes for Update/reinstall of notepad++

Switch tab
    Setting -> Preferences -> Document List Panel -> Show

Change code brace style
    Setting -> Style Configurator -> Global Styles -> Brace highlight style -> reset the brace display style here

Enable left explorer panel 
    View -> Folder As WorkSpace -> (turn on this option to enable the left panel)

Workspace panel    
    View -> Project -> Project Panel 1 -> project panel (ref: http://docs.notepad-plus-plus.org/index.php/Project_Management)

If mistakenly close the left panel we need to switch off/on the following option to reenable the Document list panel)
    Setting -> Preferences -> Document List Panel -> Show

Style tabs
    Settings -> Style Configurator -> Global Styles -> Active tab focused/Acitve tab unfocused/Acitve tab text/Inactive tabs
